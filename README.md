# Python Programming Quiz Application 🐍

A simple and interactive console-based Quiz Application developed using Python. This project is designed to test users' knowledge of Python programming concepts through 20 multiple-choice questions. The application presents one question at a time, accepts user input, validates answers, and calculates the final score and percentage at the end of the quiz.

The project demonstrates the practical implementation of fundamental Python concepts such as dictionaries, lists, loops, conditional statements, variables, user input handling, string methods, and arithmetic operations. Each question contains four options (A, B, C, and D), and the correct answers are distributed across all options to make the quiz more engaging and less predictable.

## Features

* 20 multiple-choice questions based on Python programming.
* Interactive command-line interface.
* Four options (A, B, C, and D) for every question.
* Immediate feedback for correct and incorrect answers.
* Displays the correct answer when the user makes a mistake.
* Calculates and displays:

  * Total Questions
  * Correct Answers
  * Wrong Answers
  * Final Score
  * Percentage
* Beginner-friendly code written using only basic Python concepts.

## Learning Outcomes

This project helped in understanding how different Python concepts work together to build a real-world application. It improves logical thinking, problem-solving skills, and provides hands-on experience with user interaction and scoring logic in Python.

This project is ideal for beginners, BCA/MCA students, and anyone who wants to practice Python fundamentals by building an interactive and practical console application.
